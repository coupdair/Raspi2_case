# RPi4_case

Raspberry Pi4 case in openscad

3D printable Raspberry Pi case.

![3D view](3Dview.png)

# design

`make design`

![TOP](box_TOP.png)

![bottom](box_bottom.png)

# print

## all

make both top and bottom STL files

`make`

## TOP

make only top STL file

`make top`

## bottom

make only bottom STL file

`make bottom`
